############# README ##############

### MALOT Jordan & BOTELLA Benjamin

Pour notre site, nous avons travaillé à partir d'un template fabriqué avec les librairies CSS Twitter-Bootstrap.
Le site est entièrement Responsive.

Les fichiers HTML sont conformes aux normes W3C.
Notre fichier CSS est conformes aux normes W3C (app.css).
Cependant, nous avons passé le CSS de bootstrap au W3C validator CSS et nous étions étonné du résultat, qui affiche
des erreurs.

Pour notre site, nous avons choisi de réaliser la promotion d'un festival fictif se déroulant dans Nancy.
Nous nous sommes inspirés de site évoquant le même sujet (les festivals) comme le site du festival du
chien à plumes ou encore du cabaret vert. Nous avons rédigé certains contenu, d'autres sont repris de ces sites que
nous avons analysé au préalable. Toutes les photos et les images sont également issues de ces festivals.
